OCR Segmentation Micro-Service


Goal:

    Extract the individual line images from the binarized image.

Usage:

    1. Call OCR segmentation micro-service with url: "http://10.5.146.92:8002/segmentationapi".

    2. Input: the path of the binarized image(s), and uploaded in form style. Usually the images are the output of OCR binarization service with extension ".bin.png".

    3. Output: a zip file containing equal number of folders to the number of input images. Each folder contains the extracted line images. Usually they are {imagename}/0100??.bin.png

    4. E.g.
        For one image: curl -F "file=@path_to_image" -o name_of_output_file http://10.5.146.92:8002/segmentationapi
        For two images: curl -F "file1=@path_to_image1" -F "file2=@path_to_image2" -o name_of_output_file http://10.5.146.92:8002/segmentationapi

Note:

    The image(s) must be the binarized image(s).
